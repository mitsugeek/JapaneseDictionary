### ゃ

「や」行の仮名の一つ。五十音図では「や・ゆ・よ」と並び、清音と濁音の両方が存在する。

#### 1. 清音

1. 「きゃ」：「き」 + 「や」
2. 「しゃ」：「し」 + 「や」
3. 「ちゃ」：「ち」 + 「や」
4. 「にゃ」：「に」 + 「や」
5. 「ひゃ」：「ひ」 + 「や」
6. 「みゃ」：「み」 + 「や」
7. 「りゃ」：「り」 + 「や」

#### 2. 濁音

1. 「ぎゃ」：「ぎ」 + 「や」
2. 「じゃ」：「じ」 + 「や」
3. 「ぢゃ」：「ぢ」 + 「や」
4. 「びゃ」：「び」 + 「や」
5. 「ぴゃ」：「ぴ」 + 「や」

#### 3. 促音

促音「っ」を伴う場合もある。

1. 「きゃっ」
2. 「しゃっ」
3. 「ちゃっ」
4. 「にゃっ」
5. 「ひゃっ」
6. 「みゃっ」
7. 「りゃっ」
8. 「ぎゃっ」
9. 「じゃっ」
10. 「びゃっ」
11. 「ぴゃっ」

#### 4. 用法

「ゃ」は主にカタカナ語のローマ字表記に使用される。また、一部の漢字の読みにも含まれる。
