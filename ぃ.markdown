# 「ぃ」

「ぃ」は、日本語の仮名の一つである。五十音図では「い」の右上に書かれる。

## 使い方

「ぃ」は、主に下記のような用途で使われる。

- 「い」を小さく表記する時に、「ぃ」を使う
- 外来語において、単語の末尾に「ぃ」をつけて日本語の発音に合わせることがある

## よく混同される文字との違い

「ぃ」は、長音記号や小文字「つ」と混同されることがあるが、それぞれ異なる役割を持っている。

- 長音記号は母音を長く伸ばすために使われる
- 小文字「つ」は、次の文字の発音を促すために使われる

## 例

- 「すずしい」→「すずしぃ」
- 「ハッピィバースデー」
